# Session Overview

## Goal

Add a `Taskfile.yml` with standard `build`, `test`, `lint`, and `clean` tasks that
follow the repository's detected language/tooling.

## Success Criteria

- `Taskfile.yml` exists and routes Node projects through the repo's package scripts.
- The tasks are usable for this handbook repository without introducing a new toolchain.
- The change is committed, pushed, and merged through a PR.
